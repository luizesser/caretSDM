# Bioclimatic Variables

A `stars` object with bioclimatic variables (bio1, bio4 and bio12) for
the Parana state in Brazil. Data obtained from WorldClim 2.1 at 10
arc-min resolution.

## Usage

``` r
bioc
```

## Format

\## \`bioc\` A `stars` with 1 attribute and 3 bands:

- bio1:

  Annual Mean Temperature

- bio4:

  Temperature Seasonality

- bio12:

  Annual Precipitation

## Source

\<https://www.worldclim.org/\>
