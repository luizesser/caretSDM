# Araucaria angustifolia occurrence data

A `data.frame` object with Araucaria angustifolia occurrence data
obtained from GBIF and filtered with Parana state `sf`.

## Usage

``` r
occ
```

## Format

\## \`occ\` A `data.frame` with 420 rows and 3 columns (EPSG:6933):

- species:

  Species name

- decimalLongitude:

  Longitude in meters

- decimalLatitude:

  Latitude in meters

## Source

\<https://www.gbif.org\>
