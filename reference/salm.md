# Salminus brasiliensis occurrence data

A `data.frame` object with Salminus brasiliensis occurrence data
obtained from GBIF and filtered with Parana state `sf`.

## Usage

``` r
salm
```

## Format

\## \`salm\` A `data.frame` with 46 rows and 3 columns (EPSG:6933):

- species:

  Species name

- decimalLongitude:

  Longitude in meters

- decimalLatitude:

  Latitude in meters

## Source

\<https://www.gbif.org\>
